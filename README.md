# Проєкт DigiUni Registry

Це консольний застосунок розроблений на мові програмування Java. Керування відбувається за рахунок введення чисел/слів до консолі

При запуску програми користувач потрапляє в меню входу, де має опцію ввійти в аккаунт (1), або вийти з програми (2).
Для входу необхідно ввести логін та пароль, які зберігаються в файлі `users.json`. 
У разі успіху користувач потрапляє в головне меню, де є опції Створення (1), Пошук (2), Редагування (3), Видалення (4),
Звіт (5), Управління користувачами (6), Назад (7), Вихід (8). Доступ до керування реєстром залежить від ролі (Користувач,
Менеджер, Адмін, Заблокований). Залежно від обраної опції ви маєте можливість керувати сутностями програми (Студентами, Викладачами,
 Кафедрами та Факультетами). Адмін може змінювати Користувачів. Після завершення програми створені Студенти зберігаються до файлу `students.csv`.
